# data-analysis-portfolio

This repository contains my data analysis projects created using Python and Jupyter Notebook. It showcases my work in data cleaning, exploratory data analysis (EDA), feature engineering, visualization, and insight generation across different datasets.

## About This Repository

The goal of this portfolio is to demonstrate practical data analytics skills through hands-on projects. Each notebook includes data preprocessing, analysis, visualizations, and observations derived from the dataset.

## Skills Demonstrated

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Data visualization
- Insight generation
- Working with structured datasets in Jupyter Notebook 

## Tools and Libraries

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Excel
